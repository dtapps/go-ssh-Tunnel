## 下载使用

- 把.example.config.yaml文件重命名为config.yaml
- 修改对应的配置，运行即可

## 扩展包使用

```go
go get -u github.com/dtapps/dtapps/go-ssh-tunnel
```

```go
package main

import (
	"github.com/dtapps/dtapps/go-ssh-tunnel"
)

func main() {
	Tunnel("root", "", ":22", ":3306", "localhost:13306")
}
```
